# Cadence — SaaS Landing Page

A high-converting landing page for a fictional workflow-automation SaaS, built as
part of the [SEIDO](https://github.com/dzaimov01/seido) portfolio.

**Live demo:** _added after deploy_

## Features

- Hero with animated product/automation mockup
- Logo trust strip
- **Bento feature grid** with scroll-triggered animated stat counters
- **Pricing** with monthly/annual toggle and live price recalculation
- Testimonial + gradient CTA
- Dark, modern aesthetic: gradient mesh, glassmorphism, grid overlay, Sora +
  Manrope

## Tech

React 18 · Vite 5 · Tailwind CSS 3 · Framer Motion · Lucide

## Development

```bash
npm install
npm run dev
npm run build
npm run preview
```
